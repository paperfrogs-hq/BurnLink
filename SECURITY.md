# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main) | yes |

## Reporting a Vulnerability

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Report privately via:
- Email: hello@paperfrogs.dev
- GitHub Private Advisory: https://github.com/Joy-Majumder/BurnLink/security/advisories/new

**Response time:** We aim to respond within 72 hours and resolve
critical issues within 7 days.

## Scope

In scope: burnlink.page, all subdomains, this repository's code  
Out of scope: Social engineering, DoS/DDoS, third-party services

## Recognition

Confirmed vulnerabilities are credited in our
[Hall of Fame](https://burnlink.page/hall-of-fame).
